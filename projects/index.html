Regular projects

Stuff that I have finished and projects I'm currently working on (or planning to in the near future).

  • mumble-asterisk-bridge (Rust) - Bridges Asterisk PBX channels with Mumble voice channels, allowing for interesting setups, like calling into Mumble voice channels from landline phones.
  • ovhsms (Rust) - Library for sending SMS messages through OVH's http2sms API.
  • qbin (Rust) - Like termbin.com, but done my way.
  • dn42wiki-ng (Zola) - A rewrite of the original gollum dn42 wiki to Zola, for better performance and appearance.
  • uberbot (Rust) - IRC bot that does... I don't even know what anymore. Originally intended to replace the 6 bots on karx's IRC server with one bot.
  • SMSBot (Java) - Android command chatbot (kinda like Discord bots) that works over SMS.
  • algcalc (C/C++) - The firmware for a "gaming calculator" that I've built with my school friend.
  • IRCrarria (C#) - IRC bridge for Terraria (TShock plugin).

Discord abuse

  • chuncord (Rust) - Abuse Discord's 25MB upload limit by splitting the file in chunks.
  • ZDIMGUPMobile (Java) - Android app that uploads files to Discord through a webhook and then gives you the direct hotlink. Originally intended for sending images in IRC channels.
  • windimgup (C/C++) - Win32 version of the above.
  • dimgup3 (Rust) - Web version of the above.
  • zdiu (Rust) - GTK3 version of the above.

Toys

Very small projects written out of boredom, or to prove a point.

  • sntp-yield (Rust) - Super accurate clock, thanks to Google's NTP service.
  • lemontimer (x86 Assembly) - Desktop clock for Windows. Written in ASM while waiting in an Overwatch DPS queue.
  • binshclock (C) - Desktop binary clock for Windows.
  • defralloc (C) - Defragmenting memory "allocator". Don't even bother.

R.I.P

Stuff that's not functional and not developed anymore (most likely because interest has been lost).

  • spotk (Rust) - CLI tool for retrieving song metadata from Spotify's API. The project unfortunately died due to anti-competitive API changes introduced by Spotify.
  • uonet-rs (Rust) - Library for interacting with my school's e-journal app.
  • vidns (Rust) - PowerDNS Frontend for use in small networks to allow users to create their own domains and manage their DNS records.
  • dsuft (C) - Simple file transfer utility for *nix OSes.
